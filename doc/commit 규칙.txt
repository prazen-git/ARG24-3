커밋 메시지 유형
기능 개발 : [feature development]
기능 삭제 : [feature delete]
기능 변경 : [feature modify]
최적화     : [feature optimization]
버그 수정 : [bug fix]
리팩토링 : 코드 리펙토링 [refactoring]
형식 : 코드 형식, 정렬, 주석 등의 변경 [form]
테스트 : 테스트 코드 추가, 삭제, 변경, 수행 등 [test]
문서 : 문서 추가, 삭제 변경 : [document]
프로젝트 : 프로젝트 관리 측면에서 모든 변경 ex) 빌드 스크립트 수정, Git설정, 패키지 배포 설정 변경 등 [project]
버전 업데이트 : [version update]
충돌 수정 : [resolve conflict]
기타 : [etc]

comment 형식
[commit type] user, date (yy.mm.dd), description
ex) [feature development] mspark, 24.08.05, Add SeeYA OLED  0.6' Panel
